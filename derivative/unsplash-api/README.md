# Unsplash Image cloud function

Cloud function that uses Unsplash API to get a random image based on a keyword.

## Step by step instructions for using this derivative function

- Listener field(s) type: Short Text field
- Output field type: Image field
- To use this API, follow the steps on the [Getting Access Page](https://unsplash.com/documentation#getting-started) and set up an API key.
- Store the API key in the Google Cloud Secret Manager with the name `unsplash`.
